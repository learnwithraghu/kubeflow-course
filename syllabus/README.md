# Kubeflow Student Syllabus

A beginner-friendly Kubeflow course with 7 sections covering fundamentals to MLOps.

## Quick Start

1. **Section 1**: Install Kubeflow on k3s
2. **Section 2**: Build your first pipeline
3. **Sections 3-7**: Progress through topics

## Course Structure

```
syllabus/
├── syllabus.md              # Main syllabus (this file)
├── README.md               # This file
├── section1-fundamentals/  # Kubeflow intro & setup
├── section2-kfp/           # Kubeflow Pipelines
├── section3-data-artifacts/ # Data management
├── section4-training/       # Training jobs
├── section5-katib/         # Hyperparameter tuning
├── section6-kserve/        # Model serving
└── section7-mlops/         # Automation & MLOps
```

- Basic Python knowledge
- Docker fundamentals
- kubectl basics
- 8GB+ RAM
